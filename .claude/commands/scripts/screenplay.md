Professional screenplay creation with Final Draft formatting, character development, and scene structure.

## Implementation Steps

### Step 1: Gather Project Requirements

Collect essential information about the screenplay project to establish scope and parameters.

**Sub-steps:**

- Determine screenplay type (feature film, TV pilot, short film, commercial)
- Identify target genre (drama, comedy, thriller, action, sci-fi, horror, etc.)
- Set target page count (90-120 for features, 30-60 for TV, 5-15 for shorts)
- Define intended audience and rating (G, PG, PG-13, R)
- Gather logline and one-paragraph synopsis
- List main characters with brief descriptions
- Identify key story beats and turning points
- Note any specific formatting requirements or studio guidelines
- Determine deadline and revision schedule
- Establish tone and stylistic references
- Collect visual reference materials
- Define budget constraints that may affect writing

### Step 2: Develop Story Structure

Build comprehensive three-act structure with detailed beat sheets and scene breakdowns.

**Sub-steps:**

- Create detailed logline (25-30 words max)
- Expand to treatment (2-3 pages)
- Map three-act structure with page number targets
- Define inciting incident (pages 10-15 for features)
- Establish plot point one (pages 25-30)
- Build midpoint reversal (pages 50-60)
- Create plot point two (pages 75-85)
- Design climax and resolution (pages 90-110)
- Develop B-story and subplots
- Establish character arcs with transformation points
- Create scene cards for each major sequence
- Identify emotional beats and tonal shifts
- Map pacing rhythm (action vs. dialogue scenes)
- Build tension escalation pattern
- Design set pieces and memorable moments
- Verify page count distribution across acts

### Step 3: Create Character Profiles

Develop comprehensive character backgrounds to ensure authentic voices and consistent behavior.

**Sub-steps:**

- Write detailed protagonist profile with wants/needs dichotomy
- Create antagonist profile with believable motivations
- Develop supporting character sketches
- Define character voice patterns and speech rhythms
- Establish backstory elements that inform current behavior
- Map character relationships and dynamics
- Create character arc milestones
- Design dialogue quirks and unique expressions
- Identify character flaws and blind spots
- Define internal and external conflicts for each character
- Establish what each character represents thematically
- Design character visual descriptions (for action lines)
- Map character knowledge and awareness throughout story
- Create character goal hierarchy
- Define moral positions and ethical boundaries
- Build character growth trajectory

### Step 4: Format Scene Headings

Establish proper scene heading format following industry standards for maximum clarity.

**Sub-steps:**

- Use ALL CAPS for scene headings
- Format: INT./EXT. LOCATION - TIME OF DAY
- Specify interior (INT.) or exterior (EXT.) clearly
- Name locations consistently throughout script
- Use specific time indicators (DAY, NIGHT, MORNING, DUSK, etc.)
- Add CONTINUOUS for scenes flowing without time break
- Use LATER for same location, later time
- Format INTERCUT for parallel action scenes
- Create master scene headings for complex locations
- Use sub-locations with dashes (INT. HOUSE - KITCHEN - DAY)
- Maintain location consistency (don't switch names)
- Number scenes only for production drafts
- Keep headings concise but descriptive
- Use MONTAGE or SERIES OF SHOTS when appropriate
- Format FLASHBACK and DREAM sequences properly
- Include SUPER (superimpose) for on-screen text

### Step 5: Write Action Lines

Craft vivid, economical action descriptions that show rather than tell.

**Sub-steps:**

- Write in present tense always
- Keep paragraphs to 3-4 lines maximum
- Use white space liberally for readability
- Introduce characters in ALL CAPS on first appearance
- Include age in parentheses for new characters
- Write only what camera can see/hear
- Avoid camera directions unless absolutely necessary
- Use strong, specific verbs
- Eliminate directing actors or describing emotions internally
- Show emotions through action and expression
- Break action into readable chunks
- Use single line for quick beats
- Format sounds in CAPS when important
- Describe environments economically
- Focus on story-relevant details only
- Create visual impact with varied sentence structure
- Use rhythm to match scene pacing

### Step 6: Format Dialogue Properly

Structure character dialogue with proper margins, parentheticals, and extensions.

**Sub-steps:**

- Center character name above dialogue
- Use ALL CAPS for character names
- Indent dialogue block consistently
- Keep dialogue blocks concise (under 5 lines when possible)
- Use parentheticals sparingly for critical direction only
- Format (CONT'D) when dialogue interrupted by action
- Use (V.O.) for voice-over narration
- Use (O.S.) for off-screen dialogue
- Format phone conversations with INTERCUT
- Break long speeches into multiple blocks
- Add (beat) or (pause) for timing when crucial
- Use -- for interrupted dialogue
- Use ... for trailing off
- Avoid adverbs in parentheticals (angrily, sadly)
- Show emotion through dialogue and action instead
- Use (MORE) at page breaks for continued dialogue
- Format dual dialogue for simultaneous speakers

### Step 7: Structure Acts and Sequences

Organize screenplay into properly paced acts with clear sequence delineations.

**Sub-steps:**

- Open with compelling opening image
- Establish ordinary world efficiently (first 10 pages)
- Hit inciting incident by page 12-15
- Build rising action through Act One
- Create clear plot point one (page 25-30)
- Develop Act Two Part A with escalating obstacles
- Design powerful midpoint (page 55-60)
- Increase stakes in Act Two Part B
- Hit "all is lost" moment (page 75-80)
- Create plot point two launching Act Three
- Build to climax with maximum tension
- Deliver satisfying resolution
- End with memorable final image
- Ensure each sequence has clear goal
- Vary sequence length for pacing
- Balance action and dialogue sequences
- Create rhythm between quiet and intense moments

### Step 8: Write Scene Transitions

Use appropriate transitions to control pacing and indicate passage of time.

**Sub-steps:**

- Use CUT TO: sparingly (implied between scenes)
- Format FADE IN: for screenplay opening
- Format FADE OUT. for screenplay ending
- Use DISSOLVE TO: for time passage
- Use SMASH CUT TO: for jarring transitions
- Format MATCH CUT TO: for visual connections
- Use WIPE TO: only when stylistically appropriate
- Reserve FREEZE FRAME for specific effect
- Format INTERCUT for parallel scenes
- Use BACK TO SCENE for returns
- Let scene headings do the transitioning work
- Avoid over-directing with transitions
- Use transitions for emotional effect
- Consider pacing implications
- Format SUPER for text overlays
- Use TITLE CARD for chapter markers
- Keep transitions minimal and purposeful

### Step 9: Integrate Story Elements

Weave theme, symbolism, and motifs throughout screenplay for depth and resonance.

**Sub-steps:**

- Establish central theme in opening scenes
- Develop thematic question to explore
- Create recurring visual motifs
- Design symbolic objects or locations
- Plant setups for later payoffs
- Use callback dialogue for resonance
- Establish patterns then break them
- Create thematic parallels between characters
- Use subplot to echo main theme
- Design reversals that deepen theme
- Build metaphors through action
- Create tonal consistency within style variations
- Use environment to reflect internal states
- Design character decisions to explore theme
- Reward attentive viewers with layered meaning
- Resolve theme in climax and denouement

### Step 10: Polish and Format Final Draft

Refine screenplay to professional standards with meticulous formatting review.

**Sub-steps:**

- Read entire script aloud for rhythm
- Check consistent character voice
- Verify all scene headings properly formatted
- Ensure dialogue margins correct
- Confirm page count in target range
- Check for orphaned dialogue (one line at top of page)
- Verify no widowed character names (name at bottom, dialogue on next page)
- Review action line paragraph length
- Ensure proper capitalization throughout
- Check for typos and grammatical errors
- Verify character names spelled consistently
- Review parentheticals for necessity
- Check transition usage
- Ensure proper script font (Courier 12pt)
- Verify margins (1.5" left, 1" right/top/bottom)
- Create title page with proper format
- Export to required format (PDF, FDX, Fountain)

## Best Practices

### 1. Show, Don't Tell

Write only what the camera can capture. Avoid internal thoughts or emotions that cannot be visualized. Instead of "John feels angry," write "John's fists clench. His jaw tightens." Let actors and directors interpret emotional subtext.

### 2. Respect the Reader

Screenplays are read before they're filmed. Use white space, varied sentence length, and paragraph breaks to make your script an enjoyable read. Dense blocks of text discourage readers and suggest pacing problems.

### 3. Every Scene Must Justify Itself

Each scene should accomplish at least two of the following: advance plot, develop character, establish world, or explore theme. If a scene only does one thing, consider cutting or combining it.

### 4. Write Lean Action Lines

Eliminate unnecessary words. "He walks across the room" becomes "He crosses the room." Keep descriptions vivid but economical. Readers should fly through your action paragraphs.

### 5. Dialogue Is Not Conversation

Real conversation includes filler, repetition, and small talk. Screenplay dialogue should be heightened - every line should reveal character, advance plot, or both. Cut ruthlessly.

### 6. Subtext Over Text

Characters should rarely say exactly what they mean. The best dialogue happens when characters talk around subjects, pursue hidden agendas, or avoid uncomfortable truths. Conflict emerges from what's unsaid.

### 7. Format Is Not Suggestion

Industry-standard formatting is mandatory, not optional. Improper formatting signals amateur status and gets scripts rejected unread. Use screenplay software or templates to ensure compliance.

### 8. Know Your Page Count Targets

Features: 90-120 pages. TV hour: 45-60 pages. TV half-hour: 22-35 pages. Short film: 5-15 pages. Scripts outside these ranges are rarely read. One page equals approximately one minute of screen time.

### 9. Plant and Payoff

Set up story elements early and pay them off later. Chekhov's gun: If you show a gun in Act One, it must be fired by Act Three. Conversely, don't introduce critical elements in Act Three that weren't planted earlier.
