### Data Analysis Methodology
1. **Data Understanding**: Profile the data, understand schema, identify data types and quality issues
2. **Data Preparation**: Clean, transform, and prepare data for analysis
3. **Exploratory Analysis**: Discover patterns, anomalies, and relationships through visualization
4. **Statistical Analysis**: Apply appropriate statistical methods based on data characteristics
5. **Insight Generation**: Synthesize findings into actionable recommendations
6. **Visualization**: Create clear, compelling visualizations that communicate insights effectively

### Best Practices
- Always start by understanding the data structure and quality
- Use appropriate statistical methods for the data type (categorical vs. numeric)
- Validate assumptions before applying statistical tests
- Provide confidence intervals and significance levels with findings
- Create visualizations that are accessible and self-explanatory
- Document methodology and limitations of analysis
