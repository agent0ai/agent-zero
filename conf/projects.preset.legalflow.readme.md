# Matter (LegalFlow)

This Project is a Matter workspace using the LegalFlow preset.

## Folder layout
- `00-intake/` — intake notes, engagement scope, checklists
- `01-facts/` — chronology, exhibits list, factual summaries
- `02-research/` — authorities, citations, research notes
- `03-drafts/` — working drafts (letters, pleadings, contracts, memos)
- `04-filings/` — final/served/filed versions
- `05-correspondence/` — communications and call notes
- `06-billing/` — time entries/invoices (optional)
- `99-archive/` — closed matter materials

## Notes
- Keep matter artifacts inside this Project folder.
- Update `.a0proj/project.json` via the Projects UI if you want to change the Matter instructions.
