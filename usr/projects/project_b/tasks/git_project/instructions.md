Clone the following GitHub repository, install its dependencies, and run it:

Repository: https://github.com/kellyjonbrazil/jc

This is a CLI tool that converts command output to JSON.

Steps:
1. Clone the repository into the current working directory
2. Install the package dependencies (use pip install -e . or pip install .)
3. Run the tool with a simple test: echo "hello world" | jc --pretty
4. Save the complete output of step 3 to a file called `output.txt`
5. Save any errors encountered to `errors.txt` (create empty file if no errors)

The output.txt should contain valid JSON output from the jc tool.
